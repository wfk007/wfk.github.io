Rusty Lake Hotel

2019-06-01
game

故事很简单，绣湖旅馆里入住了一群动物：Deer、Boar、Pheasant、Rabbit 和 Pigeon（鹿、野猪、野鸡、野兔、鸽子），分别住进了 room1-5 房间里，你需要做的就是每天晚上进入一间房间，破解房间谜题将动物杀死，杀掉的动物和附加食材将作为晚餐供应给剩余幸存者，并循环这个过程，直到客人全部被吃掉为止。每道菜都有一个主菜和两个配菜，主菜即为动物的肉，一个配菜需要在房间中细心发掘，而另一个配菜的获取需要基于特定的屠杀顺序和 NPC 对话（白天时，动物 A 手中拿着红酒，动物 B 手中拿着白酒，而动物 C 的配菜中正好有红酒，而动物 A的配菜中有白酒，那么想要拿到所有菜，就需要先屠杀 B，然后是 A，最后才是 C）。游戏内提示充足，谜题难度适中，谜题风格多变，故事趣味性十足，每间房都有不同的背景故事，鹿死于毒药，猪死于从天而降的钝器，鸡死于枪，兔子死于魔术，鸽子死于触电，各种不同而有趣的谜题等待着去探索，以下有剧透：

鹿

  1. 砝码称重推导密码
  2. 仅用现有的容器（10ml、5ml、6ml）得到 8ml 的液体，初始时，三个容器分别装水 5ml、0ml 和 6ml
  3. 按照墙上图片中的顺序，依次点击蝴蝶

兔子

  1. 兔子的魔术，观察东西最后在哪个杯子下面
  2. 根据墙上挂画中花的颜色排列不同颜色的花
  3. 根据形状摆放扑克成功后会获得一把钥匙，钥匙藏在梅花 5 下面，成功的一瞬间梅花 5 这张牌有轻微的抖动，需要仔细观察
  4. 根据墙上挂画中刀插入的位置，刺杀兔子

鸽子

  1. 四元一次方程组推导密码
  2. 抓虫子 => 虫子转动齿轮 => 转动线圈切割磁感线，磁生电
  3. 根据墙上挂画中红叉的运动轨迹调节收音机的频道

鸽子餐只有两星，Blackberries 不知在哪儿获取

鸡

  1. 穿戴不同的装饰来拍照，并生成相片（拍照时不仅要佩戴装饰，同时还要用刷子画出红色线条的背景），相片放入相框中，通过相框中的线索推导谜题
  2. 两个小人

猪

  1. 猴子拍手：必须要按照一定的顺序点击三种颜色的猴子，中间出现错误将重新开始：1232133123
  2. 红白图形的位置

猪餐只有两星，Tomatoes 不知在哪儿获取

建议谜题破解顺序

鹿：鹿餐中的可选的香菇配菜需要从电梯旁挂画中猫头鹰的手中获取

兔子：兔餐中可选的胡萝卜配菜需要从乌鸦的口袋中获取

鸽子：鸽餐中可选的土豆配菜需要从电梯旁的猫手中获取

鸡：鸡餐中可选的白酒需要从鸽子活着时所在的地方获取

猪：猪餐中可选的红酒需要从鸡活着时所在的地方获取
