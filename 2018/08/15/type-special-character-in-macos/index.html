如何在 macos 下打出特殊符号

2018-08-15
macos

晚上本来打算做几道编程题，在 vscode 上写代码调试的时候突然想把快捷方式都整理出来，比如：删除一行、向下复制一行、移动一行，于是乎我写着 command+shift+k、option+shift+方向键、control+方向键，写着写着感觉很蠢，想用符号表示这些技术符号。

查资料找到的方法 not work，室友告诉我的在他 mac 上可行的方法（option+特殊符号键）在我这 not work either，一瞬间好多想法出现在我脑海：我 mac 坏了？室友的 macos 版本太老？搜狗输入法的问题？好在最后把它试出来了：

勾选 在菜单中显示虚拟键盘及表情监视器

点击 显示表情与符号

点击 自定义列表，勾选 技术符号

然后双击图标就可以打出特殊符号了，如果觉得太麻烦，可以跟我一样在键盘设置中自定义文本，这样在输入相应 输入码 的时候就可以产生相应的技术符号：
