macos 下常用快捷方式整理（自用）

2018-08-16
macos

vscode：

  • 删除一行：⌘+⇧+K
  • 复制一行：⌥+⇧+↑
  • 移动一行：⌥+↑
  • 格式化：⌥+⇧+F
  • 弹出菜单：⌘+⇧+P
  • 关闭其他 Tab 页：⌘+⌥+T
  • 全局搜索：⌘+⇧+F
  • 弹出控制台：⌃+`
  • 删除文件：⌫+⌘

system：

  • 弹出特殊字符和表情：⌘+⌃+space
  • 任务管理器：⌘+⌥+esc
  • 锁屏：⌃+⇧+电源键

browser：

  • 弹出 Chrome 控制台：⌘+⌥+I
  • 清除浏览器缓存：⇧+⌘+⌫
  • 打开上次关闭的页面：⌘+⇧+T
