漫谈

要翻墙首先要有一个能和墙外世界通信的服务器，国外优质vps了解一下，然后在服务器上安装 vps 服务端，在需要翻墙的电脑端或手机端安装客户端，具体过程请参考资源中的文档。

我买了一年 19.9$ 的搬瓦工洛杉矶服务器，用支付宝付款，根据当天的汇率算完是 127.49￥。macos 用的是 v2rayx 客户端，iPhone 用的是 Shadowrocket 客户端，配置了服务器节点信息，此软件国区 Apple store 已经下架，美区的 Apple store 可以下载，下载的时候可以将自己的 Apple ID 转到美区，下完了再转回国区，或者嫌麻烦，单独申请一个美区 Apple ID 用于下载非国区 apps。注册的时候需要绑定美国的信用卡或者 paypal，如果 paypal 和 Apple ID 不在一个区域也不能绑定在一起。安卓客户端使用的是 V2RayNG ，手动配置节点信息，下载地址： release版，说实话，这个下载地址我找了好久才找到的(╯﹏╰)。

详细步骤请自行 Google。
