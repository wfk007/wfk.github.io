教程

Wiki 上提供了 Disqus 和 搜狐畅言 两种方式，头铁选择了英文。

step1：主页

访问主页，点击 GET STARTED

step2：注册

step3：选择功能

选择 I want to install Disqus on my site

step4：创建网站

  • Website Name 填写网站的名字，此名字会作为配置文件的 disqus_shortname 属性值
  • Category：网站的类别，我这里创建的是技术类

step5：订阅

订阅图上所示的 Free：针对小型，个人、非盈利性，不提供任何广告的网站。除了 Plus 中的 Direct Email Support，其他特性都支持。

step6：修改配置文件

修改配置文件：修改主题配置文件中的 disqus_shortname 字段为图上选中的值

step7：填写网站地址

填写 Website URL 字段，即博客的公网服务器地址，其他默认即可。

其他

修改头像

Settings->General

删除网站

Setting->Advanced->Delete site and comments
