哈，转眼间就到 12 月了，老实说，如果不是温度降了暖气开了雾霾来了雪花飘了，我可能都不知道一年快要结束。岁月催人老，时光催人前，我又大了一岁，也变得更热爱生活积极向上。本命年、滑铁卢、找工作、马拉松、脱单，以上就是我 2018 的关键词。

preview

Timeline：青岛实习 -> 春招的滑铁卢 -> 暑期的自闭 -> 秋天是收获的季节

过完年信心满满的一边工作一边准备春招，想上天想入地想下海（呸，下什么海，老夫可是正经人！），结果期待和结果大相径庭，本以为面试表现还不错的两家也没拿到实习 offer！研二待在实验室的小伙伴们收获颇丰，每天都在群里、朋友圈晒餐厅美食，晒办公环境，晒加班，晒团建，晒福利，吐槽工作，分享有趣的事；没有参加春招的同学也在暑假陆陆续续地准备着秋招提前批。而我，逐渐变得自闭、难过、自我怀疑和轻度抑郁，变得不自信。买了 ipad、买了零食，每天窗帘一拉大门一闭，除了吃饭时间都把自己关在宿舍打游戏。不愿让家人担心，我说我很好，不愿带给同学朋友负能量，我说我很充实。现在想想，也许当时只有宿管阿姨能感知到我的存在吧。除了自闭和难过，我用 hexo 搭了一个博客，部署在 GitHub 上，养成总结和写作的习惯；反思和总结大学到研究生阶段的经历和选择以及自己的成长轨迹；每天晚上坚持跑步，萌生了跑马拉松的想法。也许是不想留遗憾，8 月中旬我再一次投入到找工作中来：全身心投入复习、大量网申、乐此不疲的跑宣讲、笔试和面试（为了多一次选择的机会，我曾冒雨骑行十几公里参加面试）。我把自己网申的公司、参加的笔试、收到的面试和 offer 情况整理成了一个 schedule，每天醒来第一件事就是打开 schedule，然后有条不紊井然有序的安排这一天。最忙的时候一天有五场笔试+一场面试+一场宣讲，所以需要根据自身的实际情况和客观因素做一些取舍。晚上把一天的结果记录在 schedule 上，并根据收到的新邮件和短信对后续的行程做补充。忙碌的 9 月特别充实美好且短暂，最后结果还算不错，拿了 5 个 offer（华为、贝壳、度小满、58、CVTE），都是前端开发岗，做出了幸福的选择。

游戏

众所周知，我的 windows 玩不了游戏，它就是一块巨大的硬盘和物理解剖试验品，所以今年涉足的游戏主要是手游，而且几乎都是暑假消沉时玩的。今年竟然开始付费买游戏玩：中国式家长、信任的进化（ps：我仍然是个穷逼）。

游戏：

  • 端游：中国式家长、LOL、炉石传说

  • 手游：绝地求生·刺激战场、全军出击、荒野行动、QQ飞车、非人学园、第五人格、Dancing line、信任的进化、三国杀 online、节奏大师

赛事的话，年初看了青岛德杯现场，暑期看了炉石黄金赛西安站现场，所以什么时候能看一场 NBA 现场？
